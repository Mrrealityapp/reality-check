# Reality Check

Reality Check helps communities distinguish real from fake by providing transparent trust scores.

## Pages included
- index.html — Landing page
- privacy.html — Privacy policy
- terms.html — Terms of use
- profile.html — User profile form

## Mission
Restore trust and safety in digital interactions by making identity and credibility signals clear, fair, and accessible.

## Getting started
This is a static site (HTML/CSS/JS). No build step needed. Deploy directly from the repository root.
